MEMO SQL géoloc

1-Données géographiques SQL

Définir la position du Taj Mahal en Inde :

DECLARE @tm geography;
SET @tm = geography::Point(27.175254, 78.042166, 4326);

 Définir la position de la chapelle Notre Dame de Bon Voyage à Auriol, France :

DECLARE @el geography;
SET @el = geography::Point(5.649698, 43.368530, 4326);

 Calculer la distance entre le taj mahal et la chapelle :

SELECT @tm.STDistance(@el) AS [Taj Mahal to Notre Dame Church]
